Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model is a programming API for documents. The object model itself closely resembles the structure of the documents it models

How do you select an element from the DOM?

DOM Selectors, as the name suggests is used to select HTML elements within a document using JavaScript. There are 5 ways in which you can select elements in a DOM using selectors.

1-getElementsByTagName()

2-getElementsByClassName()

3-getElementById()

4-querySelector()

5-querySelectorAll()

What is event delegation in the context of the DOM, and why it is useful?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element. It refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. This can be done on a particular target element using the .target property of an event object.

How do you manipulate an Element's attributes and styles using the DOM?

There are multiple methods available in DOM to select the HTML elements and manipulate them as listed below:

  1. querySelector() method: The querySelector( ) method is used to select the first element that matches the specified selector provided as an argument.
  2. 1- querySelectorAll() method: The querySelectorAll() method returns a NodeList which contains all elements that match the selector within the document.
  3. 2- getElementById( ) method: The getElementId() method is used to select a single element using its Id.
  4. 3- getElementByClassName( ): The getElementsByClassName() method in JavaScript is used to select elements in the HTML document that have a specific class name.
  5. 4- getElementByTagName() method: The getElementsByTagName() method is used to select elements using the specified tag name.
  6. 5- getElementByName() method: The getElementsByName() method is used to select elements that have a specific name attribute. This method returns a NodeList of elements with the specified name.
